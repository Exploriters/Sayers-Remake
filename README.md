# 赛尔斯重置Sayers-Remake
Sayers的1.6+全DLC重置版！<br>
*注：由于C#文件的不兼容，1.5版本暂时不支持了…！*
## 概述
这是一款以**赛尔斯种族**为核心的种族MOD，以Humanoid Alien Races Framework为前置。<br>
赛尔斯是一种奇怪的、强力但是不稳定的人造生物，是植物与哺乳动物的粗暴融合体。它们被它们的创造者无情抛弃，而现在它们将踏上那不可能的归家之旅。<br>
